## ENVIRONMENT SETUP
### set up ruby (rvm) and rails

Even if you have it, this worked to update to recent version:
`gem install rails`

### set up mySQL
`gem install mysql2`

### set up node (nvm)
If nvm already installed:
`nvm install stable`
currently using node v9.11.0 (npm v5.6.0)

### set up angular CLI
`npm install -g @angular/cli`
